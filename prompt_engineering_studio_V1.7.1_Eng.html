What is a prompt?

Explanation:
Zero-shot is the simplest way to communicate with AI: just state your request clearly, like assigning a task to a colleague. Write concisely, specifically, and avoid ambiguity.
Use cases: Suitable for clear tasks such as: summarizing reports, writing emails, creating checklists, translating documents, etc.
Benefits: Saves time, requires no technical knowledge, anyone can do it.
Tip: Clearly state your goal, desired format (table, bullet points, etc.), and any input information if available.

Practical example:
Request: "Summarize the customer's email content into 3 main points, using simple language that is easy for new employees to understand."
AI result:
- The customer is satisfied with the product.
- Delivery was slower than expected.
- The shipping process should be improved.
